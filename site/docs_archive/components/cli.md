\# DIP CLI



The DIP CLI provides tools for working with Decision Integrity Protocol artifacts.



\## Commands



\### Sign



Signs a decision artifact.





dip sign decision.json





\### Verify



Verifies the cryptographic integrity of a decision artifact.





dip verify decision.json





\## Reference Implementation



The CLI implementation is available in the DIP repository:
